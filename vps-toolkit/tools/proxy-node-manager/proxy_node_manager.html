VPS 前缀清理与筛选

用于推断 VPS 前缀时删除多余前缀/后缀；支持一行一个，也支持用逗号分隔。

每个识别出来的 VPS 都会有自己的“VPS 前缀”，可以在下方分组里单独改；最终节点名默认以国家 emoji 和 VPS 前缀开头。

转发节点替换

支持直接替换 @host:port 或 scheme://host:port 的节点，也支持 SS 解码重写和 VMess JSON 重写。

Clash 输出批量属性

只影响输出格式为 Clash proxies YAML 的结果；留空或“不修改”不会改动原节点。
