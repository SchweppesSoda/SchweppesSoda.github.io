使用说明
  • 第一步：粘贴你的 443 模板和 80 模板（也可点 "Load demo" 看格式）。
  • 第二步：在「优选清单源」选要拉的文件 → 点 FETCH，自动从 GitHub 拉最新清单。如果代理拉不动，用 UPLOAD .txt 上传你下载好的文件，或直接粘贴。
  • 第三步：端口快速预设按钮一键填，或自己输。前缀决定节点名格式：前缀-host-port。
  • 第四步：GENERATE → Copy / Download。
  • 替换逻辑：只动模板 @host:port 这一段和末尾 #name，UUID/encryption/host=/path=/sni= 等参数原样保留。
